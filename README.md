# 🎲 Dice Game - Boss Level Challenge 01

🚀 Live Demo: https://rukshana-roshan.github.io/Boss-Level-Challenge-01-The-Dice-Game/

## 📌 About the Project

This is a simple **Dice Game** built using **HTML, CSS, and JavaScript**.  
The project is part of my web development journey to strengthen my understanding of DOM manipulation and JavaScript fundamentals.

The game randomly generates dice values for two players and automatically declares the winner based on the highest roll.

---

## 🎮 How to Play

- Refresh the page to start the game
- Two dice are rolled automatically (Player 1 & Player 2)
- The player with the higher dice value wins
- If both dice show the same number, the game is a draw

---

- HTML5

- JavaScript (Vanilla JS)
